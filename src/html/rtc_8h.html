◆ rtc_ih()

void() rtc_ih ( )

RTC interrupt handler.

This function is the interrupt handler for the Real-Time Clock (RTC). It reads the status from the RTC's register C and handles the appropriate interrupt based on the flags set in the register.

The function handles two types of RTC interrupts:

  1. Alarm interrupts (RTC_REG_C_AF): When an alarm occurs, it calls the rtc_alarm_ih() function.
  2. Update interrupts (RTC_REG_C_UF): When the RTC updates its current time, it calls the rtc_current_time_ih() function.
