{\footnotesize\ttfamily void() rtc\+\_\+ih (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



RTC interrupt handler. 

This function is the interrupt handler for the Real-\/\+Time Clock (RTC). It reads the status from the RTC\textquotesingle{}s register C and handles the appropriate interrupt based on the flags set in the register.

The function handles two types of RTC interrupts\+:
\begin{DoxyEnumerate}
\item Alarm interrupts (RTC\+\_\+\+REG\+\_\+\+C\+\_\+\+AF)\+: When an alarm occurs, it calls the \mbox{\hyperlink{rtc_8c_a49af62965717c7f8b18b684f84cd522c}{rtc\+\_\+alarm\+\_\+ih()}} function.
\item Update interrupts (RTC\+\_\+\+REG\+\_\+\+C\+\_\+\+UF)\+: When the RTC updates its current time, it calls the \mbox{\hyperlink{rtc_8c_a9db3d02888ce56d676eacc9e9ca5967d}{rtc\+\_\+current\+\_\+time\+\_\+ih()}} function.
