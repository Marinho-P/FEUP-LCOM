Detailed Description

Struct that represents a menu.

backgroundImg - xpm image of the background mainMenuLogo - xpm image of the logo font - xpm image of the font type - type of the menu nOptions - number of options(buttons) selectedOption - index of the selected option mouse - pointer to the mouse mouseImg - xpm image of the mouse mouseImgHover - xpm image of the mouse when it is hovering buttons - array of pointers to the buttons
